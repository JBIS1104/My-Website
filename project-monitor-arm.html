Monitor Arm.

Overview

Why it exists.

Conventional monitor arms require manual adjustment every time a user changes posture or repositions themselves. This project removes that friction entirely.

The arm identifies the user’s face via a webcam, computes the required joint angles, and drives servo motors in real time — keeping the monitor centred on the user’s face at all times.

Target applications include desk productivity, smart TVs, portable displays for accessibility, and collaborative workspace screens.

Software pipeline

From face to servo.

The system runs a continuous closed-loop pipeline: detect → compute → command → repeat.

1. Face Detection

OpenCV MediaPipe detects the user’s face in each webcam frame, returning bounding-box coordinates; the centroid is extracted for the next stage.

2. Angle Calculation

Pixel offset from frame centre is mapped to desired joint angles via inverse kinematics. Dead-band filtering prevents jitter on small movements.

3. HTTP Command

Target angles are serialised and posted to a lightweight Python HTTP server running on the microcontroller over the local network.

4. Hardware PWM

The server decodes the angles, generates hardware-accurate PWM signals, and drives the servo motors to the commanded positions.
